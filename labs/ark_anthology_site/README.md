# The Ark Anthology – Retro Monitor Site

This is a small static site representing the project bible for **The Ark Anthology**, a surreal series concept
based on Giordano Bruno's Ark allegory. It uses a retro green-on-black CRT aesthetic.

## Files

- `index.html` – overview, themes, world architecture
- `characters.html` – recurring characters (monks, animals, metaphysical figures)
- `episodes.html` – 12-episode guide
- `beats.html` – high-level beat sheets
- `future.html` – future seasons and extensions
- `css/retro.css` – shared retro-monitor styling
- `js/main.js` – page transitions and fade animations

Open `index.html` in a browser to explore.
