RSVP Labs

Tiny JavaScript testbeds for RSVP-style dynamics, wrapped in a CRT shell.

Overview

This mini-site hosts a set of interactive toys that poke at ideas from the Relativistic Scalar–Vector Plenum: curvature barriers, mediating fields, entropy sinks, and attractor switching. Each lab is deliberately simple and legible, using only vanilla JavaScript and a green-on-black monitor aesthetic.

Labs

Lab 01 – Curvature Field
Move a particle across a 2D grid with a central “well” and adjustable curvature; watch how trajectories bend and get trapped. Parameters roughly map to scalar potential and gradient descent.

> open curvature field
Lab 02 – Mediator Stratum
A 1D chain of sites with “lower,” “middle,” and “upper” strata. Tune the mediator coupling and see how signals from below do or don’t propagate upwards—an RSVP-ish caricature of Lorenzo’s role.

> open mediator stratum
Lab 03 – Entropy Sink
Track a “rebellion energy” variable over time with adjustable leakage into a hidden sink. Test how quickly apparent activity dies for different sink strengths.

> open entropy sink
Lab 04 – Attractor Switch
State evolves in a double-well potential; a single slider changes barrier height. Visualize when transitions between attractors are plausible versus impossible.

> open attractor switch
