Font-From-Grid: RSVP-Powered Evolutionary Simulations

Welcome to the Font-From-Grid repository, a lab for exploring the Relativistic Scalar-Vector Plenum (RSVP) framework through interactive simulators. These tools model emergent morphogenesis in swarms and technological civilizations.

Repository: github.com/standardgalactic/font-from-grid

Available Simulations

  • Xenobot Swarm Simulator v2.1 – Bio-inspired foraging in RSVP fields. (Direct link to HTML file)
  • Stars! RSVP Tech Tree Simulator v2.0 – Evolving 4X empires with factories and tech trees. (Direct link to HTML file)

Download and run locally, or use GitHub's raw view to open in browser. For full reproducibility, clone the repo and explore zips like xenobot_rsvp_suite_v4.0.zip.

Quick Start

git clone https://github.com/standardgalactic/font-from-grid.git
cd font-from-grid
# Open in browser: index.html or stars.html

See README.md for batch runs, analysis, and more.
