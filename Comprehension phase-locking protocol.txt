5. Controls, units, and pitfalls (concise)
Units
: report
W
Ω
W_\Omega
W
Ω
​
in joules; power in watts; PLV unitless. Always give sampling interval for power logs.
Band choices
: always report center freq and bandwidth (Hz). Different band extraction methods (FIR vs IIR) can bias phase estimates — use zero-phase filtering if you need non-biased phase (apply forward-backward filtering).
Power telemetry noise
: subtract a robust idle baseline (median of long idle window). Apply low-pass smoothing to sampled power to avoid aliasing from sampling jitter.
Confounds
: signal amplitude/power can trivially inflate PLV — always control for band power in regressions or use power-weighted PLV carefully.
Entrained but non-semantic locking
: include semantic checks — e.g., downstream task performance or human annotation to verify locking corresponds to semantic alignment rather than stimulus-locked entrainment.
Spike vs oscillatory locking
: if representations have transient spikes rather than oscillations, use event-phase locking metrics (e.g., spike-field coherence analogs), or smooth to extract a meaningful instantaneous phase.

"""
phase_lock_demo.py
Numerical illustration of the phase-locking work equation

W[phi] = γ ∫ |∇φ|² - ∫ s φ
with Euler-Lagrange equation:  -2γ ∇²φ = s

Solves the discrete Poisson system 2γ L φ = s
on a square grid using NumPy + Matplotlib.
"""

import numpy as np
import matplotlib.pyplot as plt
from scipy.sparse import diags

# ---------------- Parameters ----------------
N = 100               # grid size (NxN)
gamma = 1.0           # stiffness
dx = 1.0 / (N - 1)    # grid spacing
x = np.linspace(0, 1, N)
y = np.linspace(0, 1, N)
X, Y = np.meshgrid(x, y)

# ---------------- Source term s(x,y) ----------------
# Example: localized Gaussian "forcing" in the center
cx, cy = 0.5, 0.5
sigma = 0.08
s = np.exp(-((X - cx) ** 2 + (Y - cy) ** 2) / (2 * sigma ** 2))
